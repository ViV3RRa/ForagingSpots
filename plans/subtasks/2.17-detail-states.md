## Task

1. **Gallery buckets** (replace the current 0/1/2/3+ logic; dashed borders are 1.5px, on
   `--surface`):
   - **0 photos:** full-width 120px tile, 14px radius, dashed, camera-plus icon (26px) +
     "Tilføj foto" (Spectral 14.5px), `--muted`.
   - **1 photo:** large photo tile (flex 1.7, 14px radius, no chip) + add tile (flex 1, 12px
     radius, plus icon 20px + "Foto" Spectral 12px, `--faint`).
   - **2–4 photos:** large tile with "N fotos" count chip + column of [second photo, add tile].
     Note: the third/fourth photos are *not* tiled — the chip carries the count.
   - **5 photos (max):** large tile with chip + column of [second photo, third photo with a
     dark "+2" overlay (`rgba(20,15,8,.5)`, Spectral 600 18px white)]. No add tile.
   - General rules: add tile visible whenever count < 5 (and the user may edit); count chip
     whenever count ≥ 2; overlay "+N" = count − 3. Tiles keep opening the lightbox at their
     index; the add tile keeps triggering the existing add-image flow (edit sheet); real
     thumbnails as today (the mock's gradient fills and .72 opacities are placeholder art —
     ignore them).
2. **Sharing section** (owner-only, placed after Noter, before the offline notice/actions):
   - Header row: MonoLabel "Delt med" + right-aligned mono 11px `--faint` count ("2 personer",
     empty when none).
   - Shared users as rows: surface bg, line border, 13px radius, 34px brand-circle initial
     avatar (Spectral 600 14px), "@username" Spectral 15.5px ink, 28px ✕ remove button in
     `--faint`.
   - Empty state: dashed card, centered "Kun dig" (Spectral 14.5px ink2) + "Dette fund er
     privat. Del det med en ven for at give adgang." (12.5px muted).
   - **Always-visible input row** (replaces the current "+ Tilføj bruger" toggle): 48px surface
     field, 13px radius, "@" prefix (Spectral 16px, `--mono` color), placeholder "Del med
     brugernavn…" (Spectral 15px muted), plus a 48px brand square (13px radius) confirm button
     with plus icon. Keep validation, optimistic share/unshare, and Enter-to-submit.
   - The action row's Share icon button now scrolls to / focuses this input (design's
     `actShareClick` semantics).
3. **Offline lock:** restyle the offline notice with the amber offline palette
   (`offlineBg`/`offlineBorder`/`offlineInk` from the script — same family as the 3.2 banner;

4. **Header sync chips:** keep the pending/failed badge placement (margin-top 7 under the
