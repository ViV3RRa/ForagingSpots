## Task

1. **Toolbar:** replace the current count/`Select` row with the design's toolbar (count as mono
   label; custom sort trigger). Replace the shadcn `Select` with the design's popover: overlay
   click-to-close, panel top-right under the trigger, four options (Nyeste først / Ældste først /
   Efter type / Efter lokation → existing `SortOption` values), active row tinted + accent ✓.
   Adjust the list's top spacer to the design's new layout.
2. **Row meta chips:** render the design's chip row (margin-top 9, gap 6, wrap) when a spot is
   pending, failed, or shared. Build the shared chip ("Delt · N", `--line2` bg, share icon) here;
   keep rendering the existing `PendingSyncBadge` for pending/failed — its chip restyle is
   subtask 3.2 (don't restyle it here).
3. **Row action sheet:** replace the `DropdownMenu` with the design's bottom action sheet on the
   1.3 sheet primitive: handle, spot-name header (Danish type label), rows "Vis på kort",
   "Rediger", "Del", divider, "Slet" (accent, weight 600). Trigger becomes the design's 32px
   rounded-square ⋮ at the row's top-right (`align-self: flex-start`). Keep all existing
   behavior: owner-only edit/share/delete, "Vis på kort" for everyone, delete confirmation
   dialog, event propagation stops.
4. Row layout tweaks per the updated design: gap 14px. Keep search/empty states and distance
   display unchanged.
5. Both themes + safe areas.
